Utilizamos la tecnología de web scraping para acceder a una página que generaba informes trimestrales. La interfaz gráfica contaba con varios botones que permitían descargar la información ya filtrada según los requisitos, además de analizarla y presentar diagramas y estadísticas solicitadas. Estos resultados se descargaban en archivos CSV o en formato Parquet para su manejo en una base de datos. El sistema se generó en un archivo ejecutable (.exe) y se implementó utilizando Python.
